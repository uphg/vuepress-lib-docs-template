> 注意，请勿安装 0.0.7 及之前的版本，该版本及之前的版本均为测试发布功能所发布。

## yarn

使用 yarn 安装

```sh
yarn add -D tulip-ui
```

## npm

或者你可以使用 npm

```sh
npm install -D tulip-ui
```

## 图标

`t-icon` 组件默认支持 svg symbol 方式导入，例如：导入 svg icon 的 id 格式为 `id="icon-search"` 使用时以以下方式使用即可

```vue
<template>
  <t-icon name="search" />
</template>
```
